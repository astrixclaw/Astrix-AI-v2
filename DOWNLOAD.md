## 🚀 Quick Install (3 Steps)

### Step 1: Download
Copy the EXE file to your Windows PC:
```
Astrix AI Setup 2.0.0.exe
```

### Step 2: Install
1. Double-click the EXE
2. When Windows SmartScreen appears, click "More info" → "Run anyway"
3. Click "Install"
4. Wait 1-2 minutes for installation to complete

### Step 3: Launch
1. Click the Start Menu shortcut or Desktop icon
2. Login with:
   - **Username:** `admin`
   - **Password:** `password`
3. Click "Sign In"
